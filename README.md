# vue-templet

본 자료는 예시형 자료이며
제가 간단한 작업을 할때 주로 사용하는 폴더 구조이며
components 폴더에는 사용할 component를 추가하고
pages에는 보일 화면을 추가하시면 됩니다. 상황에따라 pages를 view , views로 표기하는곳도 많아요
css는 vuetify를 이용했기에 많이 사용하지는 않았습니다. 
궁금하신점이 있다면 불러주세요 :)
